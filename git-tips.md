### Basic ###
```
git diff            : view diff on all files
git difftool        : view diff using difftool
git merge <branch>  : merge <branch> into current branch
git rebase <branch> : pull <branch> into current by placing each commit on top of <branch>
```

### Specific ###
```
git rebase -i <commit> : enter interactive rebase mode
git difftool <branch>  : view diffs in vim (great for code review)
```
